# ☀️ SolarTrack – Solar Installation & Subsidy Management System

A web-based **Solar Installation & Subsidy Management System** developed using **Spring MVC**, **Java**, **Hibernate**, **JSP**, and **MySQL**. The application streamlines the process of solar installation requests, installer management, approval workflow, and subsidy reporting through role-based access.

---

## 📖 Project Overview

SolarTrack is designed to simplify the management of residential solar installation requests. It connects residents, installers, and administrators through a centralized platform where installation requests can be submitted, approved, completed, and monitored efficiently.

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/KarunaGirase/SolarTrack.git
```

### Import Project

1. Open Eclipse IDE.
2. Import as **Existing Maven Project**.
3. Configure Apache Tomcat Server.

### Database Setup

1. Install MySQL.
2. Create the project database.
3. Import the SQL script.
4. Update the database credentials in the Hibernate configuration.

### Run the Project

Deploy the application on Apache Tomcat and open:

```
http://localhost:8080/SolarTrack
```
